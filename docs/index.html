大规模 Kubernetes 集群诊断系统，用云原生的方式诊断大规模云原生系统

KubeProber 是一个针对大规模 Kubernetes 集群设计的诊断工具，用于在 kubernetes 集群中执行诊断项以证明集群的各项功能是否正常，采用云原生的方式来编排以及运行你的诊断项，支持用户自定义诊断项，同时依托底层的集群管理架构可实现针对大规模 Kubernetes 集群的统一运维管理

云原生 !

核心逻辑采用 Operator 来实现，提供完整的Kubernetes API兼容性
